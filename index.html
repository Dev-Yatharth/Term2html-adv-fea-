Lists

There are two types of list in html5.
  1. Ordered List(ol)
  2. Unordered List(ul)

Ordered list: It is used to display items in a specific order like the one above which is a ordered list(ol). It show a item in a list with a specific number. It is also called Numbered List

Unordered list: As the name specifies, it is not in order like this
  • a
  • b
  • c
